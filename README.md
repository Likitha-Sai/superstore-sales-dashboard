# 📊 Superstore Sales Dashboard

An interactive Sales Analytics Dashboard built using **Python, Pandas, Streamlit, and Plotly**. The dashboard allows users to explore sales data through dynamic filters and interactive visualizations, helping identify trends, regional performance, and business insights.

---

## 🚀 Features

* 📁 Upload Excel files dynamically
* 📅 Filter data by custom date ranges
* 🌍 Filter data by Region, State, and City
* 📊 Category-wise Sales Analysis (Bar Chart)
* 🥧 Region-wise Sales Distribution (Donut Chart)
* 📈 Time Series Sales Analysis (Line Chart)
* 🗺️ Treemap Visualization (Region → Category → Sub-Category)
* 🧩 Segment-wise and Category-wise Sales Distribution
* 📋 Interactive Summary Tables
* 📆 Month-wise Sub-Category Sales Pivot Table
* 🔍 Sales vs Profit Scatter Plot
* 💾 Download filtered datasets as CSV files

## ▶️ How to Run the Project

### Clone the repository

```bash
git clone https://github.com/Likitha-Sai/superstore-sales-dashboard.git
```

### Navigate to the project folder

```bash
cd superstore-sales-dashboard
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the Streamlit application

```bash
streamlit run app2.py
```
